# 🩸 Blood Donation DApp

A decentralized application (DApp) for managing blood donors and blood requests using Blockchain technology.

---

## 🚀 Features

- 🧑‍🤝‍🧑 Register blood donors  
- 📋 View list of donors  
- 🩸 Request blood  
- 📊 View blood requests  
- 🔗 MetaMask wallet integration  
- ⛓️ Blockchain-based data storage (Ganache)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Blockchain:** Solidity  
- **Web3 Integration:** Web3.js  
- **Wallet:** MetaMask  
- **Local Blockchain:** Ganache  
- **IDE:** Remix  

---
